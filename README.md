# Tanks Game

## Game Controls:
### Player 1:
- Move: WASD
- Shoot: Space bar

### Player 2:
- Move: Arrow keys
- Shoot: Enter key

## Download game
Download the game version for windows on itch.io: https://thsouza.itch.io/tanks

## Gameplay
Move around and shoot the other player to damage them. Win 5 rounds to finish the game.
